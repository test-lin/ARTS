# ARTS 挑战

## 说明

ARTS 是 Algorithm、Review、Tip、Share 的简写

1. Algorithm：每周至少做一个 leetcode 的算法题
2. Review：阅读并点评至少一篇英文技术文章
3. Tip：学习至少一个技术技巧
4. Share：分享一篇有观点和思考的技术文章
